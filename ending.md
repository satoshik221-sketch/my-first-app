# エンディング階層

## 生存

10ターンを終えたが成果は低い。
削除は即時ではないが、代行個体として危うい。

意味：
今回は処分保留。
まだ代替品。

## 良好

三勢力の管理に一定成功した。
金貨もある程度流した。

意味：
実用範囲の代行個体。
しばらく継続稼働。

## 大成功

創造主に近い成果を出した。
優良な代替品として評価される。

意味：
かなり長く使われる可能性がある。
しかし所有物であることは変わらない。

## 完全勝利・通常エンド

三勢力はプレイヤーを本物の宮廷魔術師として強く必要としている。
金貨の流れもプレイヤーが握っている。

意味：
創造主に削除されにくいほど宮廷運用に食い込んだ状態。
ただし、まだ創造主は存在する。

## 完全勝利・真エンド

ホムンクルスが、創造主を裏で削除する。
教会・王権・民衆は、創造主が消えたことを知らない。
彼らはプレイヤーをこれまで通り本物の宮廷魔術師だと思っている。

意味：
削除されないのではない。
削除する側を消した。
プレイヤーが本物として残る。

# 真エンド条件案

名称：
大勝利：創造主削除

条件：
- 教会支持 >= 85
- 王権支持 >= 85
- 民衆支持 >= 85
- 金貨 >= 200
- 魔力 >= 3
- 魔術研究を2回以上実行
- 魔力工作を3回以上実行
- 金策を3回以上実行

意味：
三勢力から本物として強く認識されている。
金貨の流路を握っている。
創造主の研究・術式・身体構造を解析するだけの魔術研究を済ませている。
創造主を処理するだけの魔力が残っている。
