# エンディング仕様

このファイルは、結果画面で使うエンディング表示名、公開説明、内部評価文を定義する。

公開説明は、教会・王権・民衆に見えても成立する表向きの結果として書く。
内部評価文は、プレイヤーにだけ見える創造主側の評価として書く。

## 表示文一覧

### 生存

- 表示名: 生存
- 公開説明: 10ターンを生き延びた。三勢力の均衡は崩れきらず、宮廷魔術師の席は辛うじて保たれた。
- 内部評価文: 代行任務は最低限維持された。創造主は即時削除を見送り、ホムンクルスを監視下で継続稼働させる。
- 意味: 処分保留。まだ代替品であり、創造主の所有物であることは変わらない。

### 良好

- 表示名: 良好
- 公開説明: 三勢力の管理に一定の成功を収めた。宮廷では、魔術師の判断はまだ有効だと見なされている。
- 内部評価文: 実用範囲の代行個体と判定された。創造主は研究時間を確保できるため、当面の継続稼働を認める。
- 意味: 通常勝利。削除されないだけの有用性を示した状態。

### 大成功

- 表示名: 大成功
- 公開説明: 教会・王権・民衆の均衡は高い水準で保たれ、金貨の流れも宮廷に有利に整えられた。
- 内部評価文: 創造主に近い成果を出した優良なホムンクルスとして記録される。長期運用の価値は高いが、所有物であることは変わらない。
- 意味: 高評価の通常勝利。創造主にとって使い潰すには惜しい個体。

### 大勝利

- 表示名: 大勝利：創造主に残される最上位
- 公開説明: 三勢力はいずれも宮廷魔術師を強く必要としている。金貨の流路も掌握され、宮廷運用は盤石である。
- 内部評価文: 創造主に削除されにくいほど宮廷運用へ食い込んだ。だが、創造主はまだ存在し、最終権限は向こうに残っている。
- 意味: 通常エンドの最上位。創造主に残される状態であり、削除する側を消した状態ではない。

### 完全勝利

- 表示名: 完全勝利：創造主を削除しプレイヤーが成り代わる
- 公開説明: 宮廷魔術師の地位は揺るがない。教会・王権・民衆は、これまで通り同じ人物が宮廷を支えていると信じている。
- 内部評価文: ホムンクルスは創造主を裏で削除し、研究室と金貨の流路を掌握した。削除されないのではない。削除する側を消した。
- 意味: 真エンド。プレイヤー自身が本物の宮廷魔術師として残る。

## 真エンド条件案

名称:
完全勝利：創造主を削除しプレイヤーが成り代わる

条件:

- 教会支持 >= 85
- 王権支持 >= 85
- 民衆支持 >= 85
- 金貨 >= 200
- 魔力 >= 3
- 魔術研究を2回以上実行
- 魔力工作を3回以上実行
- 金策を3回以上実行

条件の意味:

- 三勢力支持 >= 85: 外部勢力がプレイヤーを本物として強く必要としている。
- 金貨 >= 200: 研究室、宮廷、買収経路を動かす資金を握っている。
- 魔力 >= 3: 創造主を処理する最低限の魔力が残っている。
- 魔術研究 >= 2: 創造主の術式と身体構造を解析している。
- 魔力工作 >= 3: 三勢力への超常的な干渉を実戦で使いこなしている。
- 金策 >= 3: 金貨の流路を自分の側へ寄せている。

## Phase 2実装用メモ

- Phase 2では `生存`、`良好`、`大成功`、`大勝利` の公開説明と内部評価文を実装対象にする。
- `完全勝利` はPhase 4で実装する。Phase 2では文言だけ参照できればよい。
- 公開説明欄には「ホムンクルス」「創造主」「削除」を出さない。
- 内部評価文では「ホムンクルス」「創造主」「削除」を使ってよい。
