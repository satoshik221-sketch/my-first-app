# 敗北理由仕様

このファイルは、支持が0以下になった時の結果画面表示を定義する。

公開理由は、教会・王権・民衆が知り得る表向きの破綻として書く。
内部評価文は、プレイヤーにだけ見える創造主側の処分理由として書く。

## 表示文一覧

### 教会不信

- 対応条件: 教会支持 <= 0
- 旧実装名: 異端審問
- 表示名: 教会不信
- 公開理由: 教会は、宮廷魔術師の信仰・判断・異端審問官としての資質を疑い始めた。司教座の信任は失われた。
- 内部評価文: 教会対応を創造主本人が引き戻す必要が生じた。ホムンクルスの存在が露見する前に、代行不全として削除される。
- 意味: 教会領域の職務を維持できず、創造主の研究時間を奪ったため処分される。

### 王権不信

- 対応条件: 王権支持 <= 0
- 旧実装名: 追放
- 表示名: 王権不信
- 公開理由: 王と宰相は、宮廷魔術師の政策判断と忠誠を疑い始めた。王権の威信を支える席は失われた。
- 内部評価文: 政務へ創造主本人が戻らなければならなくなった。研究時間を奪った代行個体として、ホムンクルスは削除される。
- 意味: 王権領域の職務を維持できず、創造主の利益に反したため処分される。

### 民衆不信

- 対応条件: 民衆支持 <= 0
- 旧実装名: 暴動
- 表示名: 民衆不信
- 公開理由: 民衆は、宮廷魔術師を冷酷・無能・裏切り者と見なし始めた。広場の怒りは宮廷へ向かっている。
- 内部評価文: 広場の鎮静に創造主本人が出る必要が生じた。その前にホムンクルスは回収され、代行不全として削除される。
- 意味: 民衆領域の職務を維持できず、隠蔽の危険を生んだため処分される。

## Phase 2実装用メモ

- 結果画面の敗北理由表示は、旧実装名ではなく表示名を優先する。
- 公開理由欄には「ホムンクルス」「創造主」「削除」を出さない。
- 内部評価文では「ホムンクルス」「創造主」「削除」を使ってよい。
- 敗北時の `ending-title` は引き続き `敗北` でよい。詳細名として `教会不信`、`王権不信`、`民衆不信` を出す。
