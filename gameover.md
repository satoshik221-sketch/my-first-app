# 敗北理由

## 教会支持 <= 0

表向きの破綻名：
教会不信

外部視点：
教会は、宮廷魔術師本人の信仰・判断・異端審問官としての資質を疑い始める。
偽物だと疑っているわけではない。

内部視点：
このままでは創造主本人が教会対応に出なければならない。
創造主はホムンクルスの存在が明るみに出る前に事態を収拾し、代行不全としてホムンクルスを処分する。

## 王権支持 <= 0

表向きの破綻名：
王権不信

外部視点：
王や宰相が、宮廷魔術師本人の政策判断や忠誠を疑い始める。
偽物だとは知らない。

内部視点：
創造主が政務へ戻らなければならなくなる。
研究時間を奪われた創造主は、ホムンクルスを処分する。

## 民衆支持 <= 0

表向きの破綻名：
民衆不信

外部視点：
民衆が、宮廷魔術師本人を冷酷・無能・裏切り者と見なし始める。
偽物だとは知らない。

内部視点：
創造主が広場の鎮静に出る必要が生じる。
その前にホムンクルスは回収・処分される。
